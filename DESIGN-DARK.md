# Design System Theme

This document outlines the core visual theme of our design system, reflecting the current token values.

## Color Palette

Our theme uses a **dark** color mode, providing a sophisticated and low-light user experience.

-   **Primary Color**: `#1A73E8` - A vibrant blue, used for primary actions, interactive elements, and key branding.
-   **Secondary Color**: `#5F6368` - A supportive gray, suitable for less prominent UI elements and secondary actions.
-   **Neutral Color**: `#F8F9FA` - A very light gray, serving as a base for backgrounds and non-chromatic surfaces.

## Typography

The selection of typefaces ensures clarity and maintains a consistent brand voice across all content.

-   **Headline Font**: `Manrope` - Used for all major titles and headings, conveying a modern and clear presence.
-   **Body Font**: `Inter` - Applied to all paragraph text and long-form content, optimized for readability across various screen sizes.
-   **Label Font**: `Inter` - Utilized for labels, buttons, and form elements, ensuring consistency with body text and clear interaction points.

## Shape and Form

The aesthetic character of our UI components is defined by their corner treatments and spatial relationships.

-   **Roundedness**: `2` (Moderate) - UI elements feature a moderate level of roundedness, striking a balance between modern softness and structural clarity.
-   **Spacing**: `2` (Normal) - A normal level of whitespace is applied within and between UI elements, providing a balanced and comfortable visual density.
